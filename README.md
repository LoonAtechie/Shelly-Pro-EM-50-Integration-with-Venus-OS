# Shelly-Pro-EM-50-Integration-with-Venus-OS
Enables an inexpensive single-phase, dual channel Shelly energy meter to work as a grid and inverter meter.

This project was created 2 years ago when I discovered the Pro EM-50. It is a modification of Shelly 3EM code written by others. It uses the Shelly's Outbound Websocket service and Venus Ethernet port. Implementation is done by adding these two files to the /opt/victronenergy/dbus-shelly folder.
